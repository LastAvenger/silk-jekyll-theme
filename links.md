> **提示：**  
> 使用如下代码在 links.md 中增加一条友情链接：
>
> ```
> {% raw %}{% include link name="链接名称" link="url" desc="描述" %}{% endraw %}
> ```
>
> 在文章使用如下代码可以链接到指定的友情链接：
>
> ```
> {% raw %}{% include friend name="链接名称" %}{% endraw %}
> ```
